КАК ЗАПУСТИТЬ ПРОЕКТ QARSTYLE НА НОВОМ КОМПЬЮТЕРЕ

1. Откройте эту папку в редакторе кода (VS Code / Cursor / PyCharm).
2. Откройте встроенный терминал.

3. Создайте виртуальное окружение:
Команда: python -m venv venv
(или python3 -m venv venv, если первая не сработала)

4. Активируйте виртуальное окружение:
- Если компьютер на WINDOWS: venv\Scripts\activate
- Если компьютер на MAC/LINUX: source venv/bin/activate
(В терминале слева должна появиться надпись (venv))

5. Установите все нужные библиотеки:
Команда: pip install -r requirements.txt

6. Запустите сервер:
Команда: python manage.py runserver 8001

7. Откройте браузер и перейдите по ссылке:
http://127.0.0.1:8001

Удачи на защите! Вы всё сдадите на отлично!
